# Titanic Data Preprocessing & Visualization

## Project Overview
This project performs data cleaning, encoding, normalization, and visualization on the Titanic dataset to analyze survival factors.

## Steps Performed
- Handled missing values
- Encoded categorical variables
- Normalized numerical features
- Created visualizations
- Generated insights

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn

## Key Insights
- Females had higher survival rates
- First-class passengers survived more
- Higher fare correlated with survival
